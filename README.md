## Requisitos del Sistema

* **Java Development Kit (JDK):** 8 Update 271.
* **IDE:** NetBeans 12.0.

---

## Pasos para la Instalación

1. Descomprimir el archivo `.zip` del proyecto en una carpeta local.
2. Abrir **NetBeans 12.0**.
3. Ir a `File > Open Project` y seleccionar la carpeta del proyecto.
4. Hacer clic derecho sobre el proyecto y seleccionar **`Clean and Build`** para refrescar las dependencias y compilar las clases (o tambien se puede usar la barra superior)

---

## Ejecución

1. Localizar la clase `Main.java` dentro del paquete `vista` (Ruta: `Proyecto-SIA/src/main/java/vista`).
2. Hacer clic derecho sobre el archivo y seleccionar **`Run File`** (o presionar `Shift + F6`).
3. Al iniciar, aparecerá un cuadro de diálogo solicitando elegir entre el modo **Consola** o **Ventana**. Seleccionar la opción deseada para comenzar.
