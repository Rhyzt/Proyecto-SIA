Requisitos del Sistema:
Java Development Kit (JDK) 8 Update 271.
IDE NetBeans 12.0.

Pasos para la Instalación:
Descomprimir el archivo .zip del proyecto en una carpeta local.

Abrir NetBeans 12.0.
Ir a File > Open Project y seleccionar la carpeta del proyecto.
Hacer clic derecho sobre el proyecto y seleccionar Clean and Build para refrescar las dependencias y compilar las clases.

Ejecución:
Localizar la clase Main.java dentro del paquete vista (Proyecto-SIA/src/main/java/vista)

Hacer clic derecho sobre el archivo y seleccionar Run File (o presionar Shift + F6).

Al iniciar, aparecerá un cuadro de diálogo solicitando elegir entre el modo Consola o Ventana. Seleccionar la opción deseada para comenzar.
